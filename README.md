# Optimal Transport for linear Independent Component Analysis

This project is being done as the Master Thesis at the Eberhard Karls University of Tuebingen and Empirical Inference, Max Planck Institute for Intelligent Systems by Ashutosh Jha, enrolled in program M.Sc. Quantitative Data Science Methods.

In this project we will explore the application of Optimal Transport Methods (Wasserstein Distance) for the task of Independent Component extraction in a linear mixture setting.
